This is a Name Days monitoring app for PC.

- Implementation language: Python.
- The program run only on Windows
- Designed to run at system startup.
- the software can be start :
  - manually 
  - at Windows stratup time.
- The software conserves PC resources and uses minimal memory and CPU.
- The software is multilingual; English is the default and Hungarian is included. It should be easy to add other languages.
- The UI uses Python QT5.
- The user can edit the set of names to be notified on their name days.	
- The software sends notifications every 30 minutes.
- On the notification window the user can click: <Later>, <Mail>, <Done>.
- When <Later> is clicked the notification window closes, and a new notification will appear after 30 minutes.
- When <Done> is clicked, no further notifications will be sent for that name.
- If automated mail to a contact is enabled and the name has one or more registered email addresses, the software can send a predefined automatic email before the event. When <Mail> is clicked the software will attempt to send the email; if sending succeeds no further notifications will be shown.
- The mailing will work using gmail account.

When multiple names are registered for a single day, each name will have its own notification handling.
The software can send automated mail.

The program shall have a tray icon with a menu offering:
  - Edit notification database
  - Show all names for the current day
  - Query namedays for name, 
    - one dialog with a lineeditor
	- <Query> <Exit>
	- When Exit clicked the dialog will be closed
	- When Query clicked and the name editor not empty 
	  - if name exists in database update the record
	  - otherwise append the database
  - Configure settings
	- frequency of monitoring
  - Exit program

What data shall be registered for names to notify:
name, recipient, their email(s), prewritten email.
